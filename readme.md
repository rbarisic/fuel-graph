# Fuel-Graph
---

## Beschreibung

Eine Einfache Javascript-App, die mit Hilfe von einigen Daten einen Graphen erstellt.

Verwendete Tools:

- Jquery
- Raphael
- Morris
- PureCSS
- FontAwesome
